MATLAB GUI interface for Flow system for Optimal Control Lab (MISCIT):

_This interface uses DDE SERVER to communicate data with the PLC.

_To use the interface, you need to follow these steps carefully:

1.lunch DDE Manager (Demarrer-> OMRON -> cx-server -> DDE Manager).
 
2. Over the preview window of the new appeared icon of DDE, right click -> Projet -> ouvrir -> select "BA26" from the folder "Flow_Lab".

3. lunch FlowControl application from the folder "Flow_Lab" and do not use it (hide it in the corner)
